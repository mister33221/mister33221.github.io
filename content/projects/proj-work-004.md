## 專案背景

為不動產相關業者開發的後台管理系統，提供物件資料的新增、查詢、修改、刪除，以及基本報表功能。

## 我的角色

擔任 PG，負責後端 API 開發與資料庫設計協作。

## 主要工作

- 與 SA 協作梳理業務流程，設計資料表結構
- 實作後端 RESTful API：物件查詢、篩選、CRUD 操作
- Angular 前端介面部分功能實作

## 技術細節

- 後端：Spring Boot
- 前端：Angular
- 資料庫：PostgreSQL

## 心得

相對獨立的專案，讓我有機會從頭到尾參與資料庫設計與 API 規劃。這次更注重 API 設計的命名一致性與錯誤回應格式的標準化。
