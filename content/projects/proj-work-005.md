## 專案背景

為服飾品牌打造的電商購物平台，涵蓋前台購物流程（商品瀏覽、購物車、結帳）與後台管理（商品管理、訂單管理、會員管理）。

## 我的角色

擔任 PG，負責後端核心功能開發。

## 主要工作

- 使用 Spring Security + JWT 實作會員認證、登入登出與角色權限控管（顧客 / 管理員）
- 實作商品 CRUD、庫存管理、分類篩選等後端 API
- 實作訂單流程：建立訂單、狀態更新、歷史查詢
- 購物車邏輯實作（加入、修改數量、刪除）

## 技術細節

- 後端：Spring Boot + Spring Security + JWT
- 前端：Angular 13 + PrimeNG
- 資料庫：PostgreSQL

## 心得

這是第一次完整實作 Spring Security 的專案。JWT 的 stateless 設計讓 API 更乾淨，但需要妥善處理 Token 刷新與失效的邊界情況。這個專案讓我對認證授權流程有了更深的理解。
